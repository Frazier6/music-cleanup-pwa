How to clean up your library
6 STEPS · ABOUT 2 MINUTES
1
Select your music folder
Click the drop zone in the sidebar or drag your music folder onto it.

Supported: MP3, FLAC, WAV, M4A, AAC, OGG, OPUS, AIFF, WMA and more.
Open folder picker
2
Set your options
Auto-select duplicates pre-checks non-recommended copies — the scan picks the best quality file to keep (FLAC over MP3, higher bitrate wins).
3
Run the scan
Click Scan Library. Four stages run automatically:

Index → finds all audio files
Meta → reads tags (title, artist, album)
Hash → fingerprints every file
Dedup → groups the duplicates

Large libraries (10,000+ files) take 1–3 minutes.
4
Review duplicate groups
Groups are color-coded:

■ Exact Copy — byte-for-byte identical
■ Same Audio — same recording, different format
■ Metadata Match — same title + artist

The green Keep badge marks the recommended file.
5
Select files to remove
Check the files you want to delete. Use the group-level checkbox to select all non-kept files at once. Then click Move to Trash Review.
6
Download and run the delete list
In the Trash tab click Download Delete List — a text file with ready-to-run shell commands.

macOS / Linux: open Terminal, paste commands
Windows: open PowerShell, paste commands

The browser cannot delete files directly — this step is required.
Select a folder to begin.
