Select your music folder
Click the drop zone in the sidebar or drag your music folder onto it. Supports MP3, FLAC, WAV, M4A, AAC, OGG, OPUS, AIFF, WMA and more.
Open folder picker
2
Set options and scan
Auto-select duplicates pre-checks the lower-quality copies — picks the best file based on format and bitrate. Click Scan Library when ready.
3
Scan runs in 4 stages
Index → finds audio files · Meta → reads tags · Hash → fingerprints each file · Dedup → groups duplicates. Large libraries (10,000+ files) take 1–3 minutes.
4
Review duplicate groups
Groups are color-coded: red = exact copies · orange = same audio, different format · yellow = same title + artist. The green Keep badge marks the recommended file.
5
Select and move to trash
Check files to remove — the group checkbox selects all non-kept files at once. Click Move to Trash to stage them for deletion.
6
Download delete list
In the Trash tab click Download Delete List for a ready-to-run shell script. macOS/Linux: paste into Terminal. Windows: paste into PowerShell. Browser security prevents direct deletion.
