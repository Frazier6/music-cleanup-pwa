How to clean up your library
6 steps · takes about 2 minutes
10+
Formats
0
Uploads
1
Select your music folder
Click the drop zone in the sidebar or drag your music folder onto it. Works with any folder — your whole library or just a subfolder.

Supported: MP3, FLAC, WAV, M4A, AAC, OGG, OPUS, AIFF, WMA and more.
Click to open folder picker
2
Set your options
Auto-select duplicates pre-checks the non-recommended copies for you — the worker picks the best quality file to keep (FLAC over MP3, higher bitrate wins).

Turn on Expand groups on load if you want all groups open by default.
3
Run the scan
Click Scan Library. The scan runs in 4 stages:

Index → finds all audio files
Meta → reads tags (title, artist, album)
Hash → fingerprints every file
Dedup → groups the duplicates

Large libraries (10,000+ files) take 1–3 minutes.
4
Review duplicate groups
Groups are color-coded by match type:

Exact Copy — byte-for-byte identical files
Same Audio — same recording, different tags or format
Metadata Match — same title + artist

The green Keep badge marks the recommended file to keep. You can override it with Set Keep on any row.
5
Select files to remove
Check the files you want to delete — or use the group-level checkbox to select all non-kept files in one click. Use Select all Exact in the filter bar for bulk selection.

Then click Move to Trash Review.
6
Download and run the delete list
Go to the Trash tab and click Download Delete List. This gives you a text file with ready-to-run shell commands.

macOS / Linux: open Terminal, paste the commands
Windows: open PowerShell, paste the commands

The app cannot delete files directly — browser security requires this extra step.
